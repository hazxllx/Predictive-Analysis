# Predictive Analysis

A healthcare risk assessment system that integrates with PMS to evaluate patient risk levels and provide actionable insights.

## Features

- Patient risk assessment
- Health score calculation
- Recommendations engine
- Dashboard for patients and staff

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js / Express
- Database: MongoDB
- Deployment: Vercel + Railway

## Setup

1. Clone repo
2. Create `.env` from `.env.example`
3. Install dependencies:

```bash
npm install
cd backend && npm install
cd ../frontend && npm install
```

4. Run:

```bash
# Backend
cd backend
npm run dev

# Frontend (new terminal)
cd frontend
npm run dev
